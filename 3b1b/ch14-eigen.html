Vecteurs propres et valeurs propres

Voir la video originale (3blue1brown)

Quand une transformation deforme le plan, la plupart des vecteurs se font litteralement deplacer hors de leur droite d'origine. Mais certains, parfois, restent sagement sur leur ligne — ils ne sont qu'etires ou comprimes. Ce sont les vecteurs propres. Le facteur par lequel ils sont etires est leur valeur propre. C'est l'une des notions les plus utiles de toute l'algebre lineaire.

Regarde mentalement une rotation de 45° du plan. Chaque vecteur change de direction — aucun ne reste sur sa droite d'origine. Maintenant regarde un cisaillement, ou une mise a l'echelle non uniforme. Tu vois des vecteurs qui ne bougent pas de leur droite : ce sont les vecteurs propres.

L'equation centrale

Un vecteur v est un vecteur propre de la transformation A si l'image de v par A est sur la meme droite que v. Autrement dit, c'est un multiple scalaire de v :

$$A \, \mathbf{v} = \lambda \, \mathbf{v}$$

Le nombre λ (lambda) est la valeur propre associee a ce vecteur propre. Trois cas :

  • λ > 1 : le vecteur est etire dans le meme sens.
  • 0 < λ < 1 : il est comprime, toujours dans le meme sens.
  • λ < 0 : il est retourne (et eventuellement etire/comprime).
  • λ = 0 : il est envoye sur l'origine — c'est un vecteur du null space.
  • λ = 1 : il est laisse strictement en place.

Cas geometriques celebres

Une rotation 2D non triviale (autre que 0° ou 180°) n'a aucun vecteur propre reel : toute droite tourne avec l'espace. (En complexes, on retrouve des valeurs propres ; on en reparle a l'ep 16.)

Un cisaillement horizontal a un seul axe de vecteurs propres : l'axe horizontal, dont tous les vecteurs restent en place. Valeur propre 1.

Une mise a l'echelle uniforme par c a tous les vecteurs comme vecteurs propres, avec une seule valeur propre c.

Une rotation 3D autour d'un axe a un axe entier de vecteurs propres : l'axe de rotation lui-meme, avec valeur propre 1. Trouver cet axe quand on a juste la matrice de rotation est un exercice classique — c'est l'eigenvector de valeur propre 1.

Comment les calculer

L'equation A·v = λv se reecrit (A - λI)·v = 0, ou I est l'identite. C'est dire que v est dans le null space de A - λI.

Pour qu'un vecteur non nul soit dans le null space, il faut que la matrice A - λI soit non inversible — ce qui veut dire det(A - λI) = 0.

On obtient une equation polynomiale en λ de degre n (pour une matrice n × n). On l'appelle polynome caracteristique. Ses racines sont les valeurs propres.

Pour chaque valeur propre λᵢ trouvee, on resout (A - λᵢ I)·v = 0 pour trouver les vecteurs propres associes.

$$\text{Pour une 2} \times \text{2 } A = \begin{bmatrix} a & b \\ c & d \end{bmatrix} : \quad \det(A - \lambda I) = (a-\lambda)(d-\lambda) - bc = \lambda^2 - (a+d)\,\lambda + (ad - bc) = 0$$

Note les deux coefficients qui apparaissent : a + d est la trace, et ad - bc est le determinant. C'est l'idee de l'episode 15 — toute la machinerie eigen 2×2 se ramene a ces deux nombres.

L'eigenbasis : la base ou tout devient simple

Voici le payoff majeur. Si on peut former une base de l'espace constituee de vecteurs propres de A (« eigenbasis »), alors la matrice de A exprimee dans cette base est diagonale : ses entrees hors diagonale sont nulles, et la diagonale contient les valeurs propres.

$$A \;\text{en eigenbasis} \;=\; \begin{bmatrix} \lambda_1 & 0 & 0 \\ 0 & \lambda_2 & 0 \\ 0 & 0 & \lambda_3 \end{bmatrix}$$

Pourquoi ? Parce que les colonnes d'une matrice sont les images des vecteurs de base (ep 3), et dans l'eigenbasis, chaque vecteur de base est un vecteur propre — son image est un multiple λ de lui-meme.

L'enorme avantage : calculer Aⁿ devient trivial en base propre — la diagonale s'eleve composante par composante (λᵢⁿ). Pour revenir dans la base standard, on utilise la matrice de changement de base (ep 13).

Un vecteur propre est un axe que la transformation laisse en place, a un facteur d'echelle pres. Dans une base de vecteurs propres, la matrice devient diagonale — et tous les calculs deviennent triviaux.

Pourquoi les valeurs propres apparaissent dans tellement d'endroits

En physique, les valeurs propres sont les modes de vibration, les frequences naturelles, les niveaux d'energie. En statistique, ce sont les axes d'inertie d'un nuage de points (analyse en composantes principales). En informatique, le rang Google de chaque page web est calcule a partir de vecteurs propres d'une matrice de liens. Partout ou un systeme se decompose en « directions independantes qui s'evoluent chacune a sa vitesse propre », les eigenvectors sont la clef. Le mot allemand eigen qui veut dire « propre, intrinseque » est bien choisi : ce sont les directions intrinseques de la transformation, independantes du choix de base.

Ce qu'il faut retenir

Un vecteur propre de A est un vecteur dont l'image par A est un multiple scalaire de lui-meme : A·v = λv. Le facteur λ est la valeur propre.

On les trouve par det(A - λI) = 0 (polynome caracteristique), puis pour chaque λᵢ, on resout (A - λᵢI)v = 0.

Dans une base de vecteurs propres, la matrice est diagonale. Cela transforme tous les calculs (puissances, exponentielle de matrice, equations differentielles…) en operations composante par composante.
