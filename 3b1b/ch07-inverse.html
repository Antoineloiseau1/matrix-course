Essence of Linear Algebra · Episode 07

Inverse, column space, null space

Voir la video originale (3blue1brown)

Resoudre une equation lineaire A·x = b est l'application la plus pratique de l'algebre lineaire — toute la science et l'ingenierie en sont remplies. Cet episode raconte ce que resoudre veut dire geometriquement : trouver le vecteur qui, transforme par A, atterrit en b.

Pose le probleme dans la langue qu'on parle depuis trois episodes. A est une transformation. Elle prend chaque vecteur de l'espace et le deplace vers une nouvelle position. La question A·x = b demande : quel vecteur de depart, une fois transforme, finit a la position b ?

Cas 1 : la transformation est inversible

Si det(A) ≠ 0, la transformation ne perd aucune information — elle ne fait pas s'effondrer l'espace sur une dimension inferieure. Il existe alors une transformation inverse, notee A⁻¹, qui defait exactement ce que A a fait.

Visuel : si A tourne le plan de 90° et l'etire par 2, alors A⁻¹ retrecit par 2 et tourne de -90°. Composees, elles donnent la transformation qui ne fait rien — l'identite.

$$A^{-1} \cdot A = I$$

Et la solution de A·x = b est simple : applique A⁻¹ aux deux cotes pour obtenir x = A⁻¹·b. Geometriquement, on remonte le voyage : on part de b et on suit la transformation inverse pour retrouver le point de depart.

Quand A est inversible, la solution est unique. Pour chaque b, il y a un seul x.

Cas 2 : determinant nul

Si det(A) = 0, la transformation ecrase l'espace dans une dimension inferieure. Tout le plan se retrouve compresse sur une ligne (ou meme reduit a l'origine, en 2D). Beaucoup de vecteurs differents atterrissent au meme endroit — on a perdu de l'information.

Resoudre A·x = b devient delicat. Deux sous-cas :

  • Si b est sur la ligne ecrasee, alors il existe une infinite de x qui s'y projettent. La solution n'est pas unique.
  • Si b n'est pas sur cette ligne, alors aucun x ne s'y projette. La solution n'existe pas.

Column space : tout ce que A peut produire

L'ensemble de toutes les sorties possibles de la transformation A — tous les A·x quand x parcourt l'espace entier — s'appelle le column space (espace colonne) de A.

Pourquoi ce nom ? Parce que A·x est une combinaison lineaire des colonnes de A (cf. ep 3). L'ensemble de toutes les combinaisons lineaires, c'est exactement le span des colonnes. Le column space, c'est le span des colonnes.

A·x = b a une solution si et seulement si b est dans le column space.

Rang : la dimension du column space

Le rang d'une matrice est la dimension de son column space. Une matrice 2×2 a rang 2 quand ses deux colonnes spannent le plan (transformation inversible). Elle a rang 1 quand ses deux colonnes sont colineaires (transformation qui ecrase tout sur une ligne). Elle a rang 0 quand toutes ses colonnes sont nulles (transformation nulle).

Une matrice n×n de rang n est dite de rang plein. C'est equivalent a det ≠ 0, c'est equivalent a « inversible ».

Null space : le voile noir des entrees ecrasees

Le null space (ou noyau) de A est l'ensemble des vecteurs x qui sont envoyes sur l'origine : A·x = 0.

Si A est inversible, le seul tel vecteur est x = 0 — le null space est trivial. Mais si A ecrase l'espace, alors toute une famille de vecteurs (au moins une ligne entiere) tombe a l'origine. Ce sont eux qui forment le null space — geometriquement, l'image inverse du point 0.

Le null space sert exactement quand on resout A·x = b dans le cas non-inversible : si x₀ est une solution particuliere et n est dans le null space, alors x₀ + n est aussi solution. L'ensemble des solutions est une solution particuliere plus tout le null space.

Column space = toutes les sorties possibles. Null space = toutes les entrees qui s'effacent. Le rang est la dimension du column space. Le theoreme du rang : rang + nullite = dimension de l'espace de depart.

Le theoreme du rang en une image

Une transformation n × n a n dimensions d'entree. Soit elles produisent des dimensions de sortie (rang), soit elles sont ecrasees (nullite). Aucune dimension ne disparait sans laisser de trace : rang + nullite = n. C'est le theoreme du rang-nullite, et il est presque tautologique une fois qu'on a la lentille geometrique.

Ce qu'il faut retenir

Resoudre A·x = b, c'est demander : « quel vecteur, transforme par A, atterrit en b ? ».

Si det(A) ≠ 0, la solution est unique : x = A⁻¹·b. Sinon, soit il n'y a pas de solution (si b n'est pas dans le column space), soit il y en a une infinite (toutes les solutions = solution particuliere + null space).

Column space = span des colonnes = toutes les sorties accessibles. Rang = dimension du column space. Null space = ensemble des vecteurs envoyes sur l'origine. Theoreme : rang + nullite = dimension de l'espace de depart.
