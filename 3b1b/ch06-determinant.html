Essence of Linear Algebra · Episode 06

Le determinant

Voir la video originale (3blue1brown)

Comment mesurer l'effet d'une transformation lineaire sur les aires ? La reponse — un seul nombre qui resume tout — est l'un des plus beaux concepts du domaine. C'est le determinant : un facteur d'echelle pour les surfaces, avec un signe en bonus qui dit si l'espace s'est retourne.

Quand on applique une transformation lineaire au plan, toutes les figures se deforment : carres, cercles, formes quelconques. Une question naturelle : de combien chaque aire est-elle multipliee ?

L'observation surprenante est que ce facteur est le meme pour toutes les regions. Si la transformation double l'aire du carre unite, alors elle double aussi l'aire d'un cercle, d'un triangle, d'une forme quelconque. C'est une propriete des transformations lineaires : elles agissent de maniere uniforme sur l'espace.

L'aire du parallelogramme image

Le moyen le plus simple de mesurer ce facteur consiste a regarder le carre unite — celui dont les cotes sont î et ĵ, d'aire 1. Apres transformation, ce carre devient un parallelogramme dont les cotes sont les deux colonnes de la matrice.

L'aire de ce parallelogramme est, par definition, le determinant de la matrice — note det(M). Puisque l'aire de depart valait 1, ce nombre est aussi le facteur par lequel toutes les aires se multiplient.

Trois cas a connaitre

Determinant positif (et grand). Les aires sont etirees. Plus le nombre est grand, plus l'etirement est fort.

Determinant entre 0 et 1. Les aires sont compressees, mais l'orientation est preservee. La transformation rapproche les points.

Determinant negatif. L'espace s'est retourne — on parle de changement d'orientation. Imagine prendre la feuille du plan et la retourner pour montrer son verso : ĵ se retrouve a droite de î au lieu d'etre au-dessus. La valeur absolue donne toujours le facteur d'echelle, mais le signe negatif raconte cette inversion. Une reflexion (symetrie miroir) est l'exemple paradigmatique d'une transformation a determinant negatif.

Determinant nul. La transformation ecrase l'espace dans une dimension inferieure. En 2D, le plan est compresse sur une ligne (ou meme reduit a l'origine). Toutes les aires deviennent 0 — d'ou det = 0. C'est le cas critique : aucune transformation a det = 0 n'a d'inverse, parce qu'on a perdu de l'information de maniere irreversible.

det(M) est le facteur par lequel la transformation multiplie les aires. Negatif = retournement de l'espace. Nul = ecrasement irreversible.

La formule 2×2, sans mystere

Pour une matrice [[a, b], [c, d]], le determinant est :

$$\det = a \cdot d - b \cdot c$$

Pourquoi cette formule ? Visualise le parallelogramme dont les cotes sont [a, c] et [b, d]. Si b = c = 0, le parallelogramme est un rectangle d'aire a · d. Le terme b · c corrige l'aire quand le parallelogramme est cisaille — c'est exactement la portion qu'on a en trop ou en manque par rapport au rectangle.

En 3D : un facteur de volume

En 3D, le determinant d'une matrice 3×3 mesure le facteur par lequel les volumes sont multiplies. Le cube unite (volume 1) se transforme en parallelepipede dont le volume est |det(M)|. Le signe (positif/negatif) reflete l'orientation du repere (droite ou gauche, regle de la main droite).

Une propriete cle

Le determinant a la propriete remarquable suivante :

$$\det(A \cdot B) = \det(A) \cdot \det(B)$$

Geometriquement, c'est evident : la composition de deux transformations multiplie successivement les aires par det(B), puis par det(A). Le facteur total est le produit.

Pourquoi det = 0 condamne l'inverse

Si une transformation ecrase tout le plan sur une ligne, on a litteralement perdu une dimension d'information. Plusieurs vecteurs differents finissent au meme endroit — donc il est impossible de remonter de l'image au vecteur de depart. L'inverse n'existe pas. det = 0 est le signal de cette perte irreversible. On y reviendra a l'episode 7.

Ce qu'il faut retenir

Le determinant d'une matrice est le facteur par lequel sa transformation multiplie les aires (2D) ou volumes (3D). Negatif : l'orientation s'est inversee. Nul : l'espace s'est ecrase dans une dimension inferieure — la transformation n'est pas inversible.

Pour une 2×2 : ad - bc. La propriete det(AB) = det(A)·det(B) est geometriquement evidente.
